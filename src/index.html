O que é esse sistema?

Esse sistema tem como objetivo auxiliar alunos de exatas com os trabalhos do seu curso. Atualmente, contém funções para os assuntos: Física Experimental , Calculo Numérico e Circuitos Elétricos, basta escolher qual você precisa no menu acima.

O sistema é completamente open source e está aberto para sugestões de melhorias, os links para os repositórios no GitHub estão presentes no rodapé.
